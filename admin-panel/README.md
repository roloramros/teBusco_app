# Te Busco Admin Panel

Panel de administración web para la plataforma de transporte Te Busco.

## Requisitos
- Node.js 18+
- API Te Busco corriendo (puerto 8004)

## Instalación
1. `cd admin-panel`
2. `npm install`
3. Configurar `.env` con la URL de la API

## Desarrollo
`npm run dev`

## Producción
1. `npm run build`
2. Servir la carpeta `dist/`
